# Action Columns Inventory

The following list maps all the Vue components where an "Actions" (or "Action") column is defined inside Data Tables, or where implicit inline action buttons are utilized for data lists.

## Administration Module
- `resources/js/pages/Admin/Users/Index.vue` (Table column)
- `resources/js/pages/Admin/Notices/Index.vue` (Table column)
- `resources/js/pages/Admin/Backup/Index.vue` (Table column)
- `resources/js/pages/Admin/ActivityLogs/Index.vue` (Table column)
- `resources/js/pages/Admin/Calendar/Index.vue` (Inline action buttons for events list)
- `resources/js/pages/Admin/Profile/Index.vue` (Card Actions / Form Actions)
- `resources/js/pages/Admin/Dashboard.vue` (Quick Actions section)
